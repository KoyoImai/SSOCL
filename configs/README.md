# ディレクトリ概要
学習・評価のハイパラなどを設定するファイルを配置するディレクトリです．
このリポジトリで回すプログラムの設定は，基本的にhydraを使用し`xxx.yaml`で管理しています．

## プログラムの全体像
このディレクトリの全体像は以下の通りです．
```
configs/
├── debug           : debug用のconfigファイルを配置
├── default         : default実行のconfigファイルを配置
└── xxx             : （今後追加予定）
```
